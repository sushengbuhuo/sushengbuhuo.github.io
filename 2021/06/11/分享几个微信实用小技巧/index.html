分享几个微信实用小技巧

发表于 2021-06-11 | 次
| 字数统计: 873 字 | 阅读时长 ≈ 3 分钟

苏生不惑第253 篇原创文章，将本公众号设为星标，第一时间看最新文章。

最近微信更新了8.0.6版本https://weixin.qq.com/ ，有个炸屎特效简直了。

还不如电脑端刷朋友圈这个功能实用，据说单向检测删除好友功能在开发中，期待下。

之前分享过不为人知的几个微信实用技巧，今天继续分享几个微信实用技巧。

语音自动转文字

上班的时候收到一大段语音，挨个点转换是不很麻烦，现在电脑端可以设置自动转文字了。

如果想导出语音见之前文章如何导出你的微信语音

转发语音

先收藏语音，进入我的收藏转为笔记，然后发送给朋友或分享朋友圈。

聊天记录合并

电脑端直接复制多条聊天信息，然后粘贴到聊天框即可合并聊天记录。

如果想看撤回消息见之前文章你刚才微信上撤回了什么？我都看到了

取消授权应用

在微信设置/隐私/授权管理里可以看到你的微信授权了哪些应用APP获取你的微信朋友关系，比如之前我写这篇文章玩转微信读书授权的微信读书 。

QQ的话可以查看这个QQ互联 https://connect.qq.com/ ，另外https://id.qq.com/可以查到你 QQ 的使用时间。

桌面快捷功能

安卓版微信新增了桌面快捷功能，长按微信图标会弹出快捷功能菜单，比如我的二维码、收付款和扫一扫，这是我最近才发现的。

微信修复工具

搜索框输入:up进入微信的修复工具，可以修复聊天记录和搜索故障等。

导出表情包

表情包只能添加到表情，如果想导出来可以使用网页版微信https://wx.qq.com/ ，之前分享的一键生成你的微信社交数据报告也是基于网页版微信，不过现在很多微信号已经不能登录，还好我的还可以用。


当然如果你有公众号可以在后台下载，关于公众号更多技巧见之前文章玩转公众号。

视频号下载

之前分享过2021年4月如何下载微信视频号的视频？简直不要太简单，不过现在新版微信去除了查看源代码功能，还好可以用抓包工具fiddler https://www.telerik.com/download/fiddler ，先在options中设置抓取 https 。

然后电脑端打开视频号，可以抓到视频播放地址域名channels.weixin.qq.com。


复制到浏览器打开视频，用idm下载就行了。


如果嫌麻烦可以借助这个工具（在公众号后台回复 视频 获取）,打开软件后点击开始获取，电脑端播放视频后就可以下载了。

微信聊天记录备份

这是个很实用的功能，现在有免费工具了 http://wxbackup.imxfd.com/ ，它支持 Windows 和 Mac系统，可以将微信聊天记录导出生成 HTML 网页，文字、语音、表情、图片、视频就能在电脑上查看了，前提需要一个苹果设备（iPhone 或 iPad）。
