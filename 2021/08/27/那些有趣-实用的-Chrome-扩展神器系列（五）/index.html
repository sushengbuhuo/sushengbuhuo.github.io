上网时间统计

这个扩展可以统计你每天打开网站的比例，扩展地址https://chrome.google.com/webstore/detail/timer-running-browsing-ti/dkdhhcbjijekmneelocdllcldcpmekmm?hl=zh-CN

Markdown Editor

之前文章分享过markdown编辑器新买了台笔记本电脑，分享些实用的Windows软件，有了这个扩展就可以在浏览器写markdown了，扩展地址https://chrome.google.com/webstore/detail/markdown-editor-for-chrom/dkpldbigkfcgpamifjimiejipmodkigk/，另外分享个markdown迁移工具https://markdown-here.com/以及在线阅读markdown https://chrome.google.com/webstore/detail/markdown-reader/gpoigdifkoadgajcincpilkjmejcaanc

草料二维码

这是草料二维码网站http://cli.im/ 的扩展，提供简单、专业、快捷的二维码生成和解码功能，扩展地址
https://chrome.google.com/webstore/detail/%E8%8D%89%E6%96%99%E4%BA%8C%E7%BB%B4%E7%A0%81-%E5%BF%AB%E9%80%9F%E7%94%9F%E7%A0%81%E5%92%8C%E8%A7%A3%E7%A0%81%E5%B7%A5%E5%85%B7/moombeodfomdpjnpocobemoiaemednkg?hl=zh-CN

沙拉查词

这是一款专业划词翻译扩展，为交叉阅读而生，大量权威词典涵盖中英日韩法德西语，支持复杂的划词操作、网页翻译、生词本与 PDF 浏览，扩展地址
https://chrome.google.com/webstore/detail/%E6%B2%99%E6%8B%89%E6%9F%A5%E8%AF%8D-%E8%81%9A%E5%90%88%E8%AF%8D%E5%85%B8%E5%88%92%E8%AF%8D%E7%BF%BB%E8%AF%91/cdonnmffkdaoajfknoeeecmchibpmkmg?hl=zh-CN

追剧助手

这个扩展可以方便的管理自己喜爱的影视剧，点击扩展图标即可查看所有剧集列表，一键跳转到播放页面播放,打开腾讯视频，爱奇艺视频，哔哩哔哩，在播放页面点击右上角的追剧按钮即可使用，扩展地址https://chrome.google.com/webstore/detail/%E8%BF%BD%E5%89%A7%E5%8A%A9%E6%89%8B/ilenakcpbigjcnneabfbjbbldjjbpcie?hl=zh-CN

Instant Data Scraper

之前写过一篇抓取数据的扩展文章不用写代码，Chrome 扩展神器 web scraper 抓取知乎热榜/话题/回答/专栏，豆瓣电影，这个扩展抓取数据更简单了https://movie.douban.com/top250，扩展地址
https://chrome.google.com/webstore/detail/instant-data-scraper/ofaokhiedipichpaobibbnahnkdoiiah

图片助手

这是一款用于嗅探、分析网页图片并提供批量下载等功能及在线收藏、检索、分享服务的浏览器扩展程序，不仅可以处理网页和本地图片，还可以批量处理，扩展地址https://chrome.google.com/webstore/detail/imageassistant-batch-imag/dbjbempljhcmhlfpfacalomonjpalpko?hl=zh-CN ，比如获取豆瓣电影top250的图片 https://movie.douban.com/top250，右键提取本页图片即可。

获取cookie

之前文章免费使用腾讯云每天定时签到京东领取京豆 获取京东cookie一直有人在后台问怎么弄，有了这个扩展就方便复制了，扩展地址 https://chrome.google.com/webstore/detail/get-cookiestxt/bgaddhkoddajcdgocldbbfleckgcbcid/related

百度文库下载

这个Chrome扩展能免费下载百度文库文档，支持导出PDF和图片，还能屏蔽文档中的广告，支持Word、PPT、PDF文档，Chrome扩展地址https://chrome.google.com/webstore/detail/%E7%99%BE%E5%BA%A6%E6%96%87%E5%BA%93%E5%85%8D%E8%B4%B9%E4%B8%8B%E8%BD%BD/imjoocoajfjgnabmlbgpcnpieibibhmd

安装扩展后打开这个文档 https://wenku.baidu.com/view/021014797dd184254b35eefdc8d376eeaeaa172f.html ，左下角有个免费导出文档按钮。

默认下载的文件是PDF格式，你也可以选择导出文本和图片格式。

Chrome清理大师

一键清理浏览器缓存和垃圾，保护隐私，让你的Chrome更快，扩展地址https://chrome.google.com/webstore/detail/clean-master-the-best-chr/eagiakjmjnblliacokhcalebgnhellfi

  • 清理浏览器缓存，cookies，网站存储和其他垃圾。提高浏览器的响应速度。
  • 清理你的密码。其他永远不会登录你的个人帐户。
  • 清理你的浏览器历史记录。其他人永远不会知道你浏览过的网站。
  • 您可以选择清理所有个人信息。您的隐私将受到保护。
  • 您的浏览器永远不会崩溃和卡住。
  • 这是Chrome商店中最有效的清理扩展。
